## How to use?

1. Download the [**`3D-GTA-Launcher.zip`**](https://github.com/KaioHSG/3d-gta-launcher/releases/latest).
2. Run **`3D GTA Launcher.hta`**.
3. Change the `Scan Folder` directory to your desired location, and then click **`Scan`**.
